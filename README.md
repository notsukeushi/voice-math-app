# こえでこたえる！さんすうアプリ

小学生向けの、音声入力で答えられる算数（足し算・引き算）のWebアプリです。

## 特徴
- 画面に大きく筆算を表示し、声で答えることができます。
- 音声認識（Web Speech API）を使用しており、ブラウザだけで動きます（Chrome または Edge 推奨）。
- 10問連続で出題され、正解・不正解を判定します。

## 使い方
1. `index.html` をブラウザ（Google Chrome または Microsoft Edge）で開きます。
2. マイクの許可を求められたら「許可する」を選択します。
3. 「🎤 こたえる (マイク)」ボタンを押して、声で答えを言います（例：「ななじゅうに」）。
4. 正解すると次の問題に進めます。
